# 🌾 Farmer Crop Management System

## 📌 Description
A Node.js and MySQL-based web application that allows farmers to log in and manage crop details like crop type, area, soil type, sowing date, and location.

---

## 🧰 Tech Stack
- Node.js + Express
- MySQL
- HTML, CSS, JavaScript
